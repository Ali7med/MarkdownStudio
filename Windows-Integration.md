# Windows Integration

> الهدف هو جعل التطبيق جزءاً من نظام Windows وليس مجرد برنامج مستقل.

---

# Default Markdown Application

يمكن للمستخدم جعل البرنامج التطبيق الافتراضي لجميع ملفات Markdown.

يدعم:

- .md
- .markdown
- .mdown
- .mkd
- .mkdn
- .mdtxt

---

# Register File Associations

أثناء التثبيت

☑ Make Markdown Studio the default application

أو من داخل Settings.

---

# Restore Defaults

زر لإلغاء ربط الملفات وإرجاع الإعدادات الافتراضية.

---

# Open With Integration

إضافة البرنامج إلى

Open With

داخل Windows.

# Context Menu

إضافة أوامر داخل قائمة الزر الأيمن.

Markdown Studio

├── Open
├── Open in New Window
├── Compare Files
├── Preview
├── Convert to HTML
├── Convert to PDF
├── Copy Markdown Path

# Explorer Preview

إضافة إمكانية معاينة ملفات Markdown مباشرة من مستكشف الملفات (ميزة مستقبلية).

---

# Explorer Thumbnail

إنشاء Thumbnail مخصص لملفات Markdown (ميزة مستقبلية).

---

# File Icon

استخدام أيقونة احترافية خاصة بجميع ملفات Markdown المرتبطة بالبرنامج.

---

# Drag & Drop

دعم سحب الملفات إلى نافذة البرنامج.

---

# Multiple File Drop

---

# Folder Drop

↓

يفتح Workspace بالكامل.
