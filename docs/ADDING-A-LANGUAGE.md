# إضافة لغة جديدة

النظام يكتشف اللغات تلقائياً من ملفات JSON. لإضافة لغة:

## الخطوات
1. انسخ ملفاً موجوداً من مجلد `lang` بجانب البرنامج (مثل `en.json`) وسمِّه برمز اللغة:
   `fr.json` · `de.json` · `es.json` …
2. عدّل الكتلة `meta`:
   ```json
   "meta": { "name": "Français", "code": "fr", "direction": "ltr" }
   ```
   - `name`: الاسم المعروض في قائمة اللغات.
   - `code`: رمز اللغة (يُطابق اسم الملف).
   - `direction`: `ltr` أو `rtl` — يضبط اتجاه النوافذ والقوائم تلقائياً.
3. ترجم قيم `strings` (اترك المفاتيح كما هي).
4. أسقط الملف في مجلد `lang` بجانب التنفيذي.
   → تظهر اللغة تلقائياً في قائمة **اللغة** عند التشغيل التالي.

## ملاحظات
- ملفات مجلد `lang` على القرص **تتجاوز** المضمّنة (تقدر تعدّل ترجمة جاهزة).
- أي مفتاح مفقود يظهر باسمه — يسهّل اكتشاف النقص.
- الأرقام تبقى لاتينيّة في كل اللغات؛ الاتجاه فقط يتبع `direction`.
