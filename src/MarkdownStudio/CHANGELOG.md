# Changelog

## [1.1.1] - 2026-07-23
### FIXED
- ألوان أقسام لوحة «ما الجديد» (جديد/تحسينات/إصلاحات) أصبحت تتبع الثيم وتتكيّف مع الوضع الفاتح/الداكن بدل ألوان ثابتة.

## [1.1.0] - 2026-07-23
### HIGHLIGHTS
- إعادة تصميم كـ«عارض أولاً»: فتح أي ملف يعرضه بمعاينة قراءة متمركزة، والمحرّر خيار بمفتاح واحد.
### NEW
- وضع عام عارض/محرّر مع مفتاح تبديل بارز، وتنضيد قراءة مريح في العارض.
- **مزامنة تمرير** ثنائية الاتجاه بين محرّر الكود والمعاينة (يتحرّكان سوية، والسطر الحالي يقابله موضعه بالعرض).
- زر «تحرير» يفتح مجلد الملف الحالي في المستكشف ويحدّد الملف.
- مثبّت Velopack مع **تحديث تلقائي** من GitHub Releases.
- ربط ملفات Markdown وتسجيل البرنامج في «التطبيقات الافتراضية»، وزر «اجعله الافتراضي لملفات Markdown».
### IMPROVED
- إعادة هيكلة الشريط: إجراءات سياقية حسب الوضع يساراً وإجراءات عامة يميناً.
- شاشة ترحيب موجّهة للعرض (فتح ملف أولاً).
- ظهور البرنامج ضمن اقتراحات «فتح باستخدام» لكل امتدادات Markdown.

## [1.0.0] - 2026-07-18
### HIGHLIGHTS
- استوديو تحرير مرئي (WYSIWYG): حرّر مباشرة على المعاينة ويتولّد كود Markdown في الجهة الثانية.
### NEW
- شريط أدوات تحرير غني: عريض/مائل/يتوسّطه خط، عناوين، قوائم، اقتباس، كود، رابط، صورة، وحجم الخط.
- جداول في الوضع المرئي مع قائمة سياق لإضافة/حذف صف وعمود.
- مزامنة المؤشّر بين المعاينة ولوحة الكود.
- ميزة **ما الجديد** ونظام سجل التغييرات.
- دعم تعدّد اللغات مع الإنجليزية واتجاه نص تلقائي (RTL/LTR).
### IMPROVED
- نوافذ التنبيه تظهر متوسّطة على نافذة البرنامج مع تخطيط سليم للأزرار.
### FIXED
- إصلاح تداخل عنوان المستكشف مع أزراره.

## [0.9.0] - 2026-07-17
### HIGHLIGHTS
- مساحة عمل كاملة على مستوى المجلد بأسلوب IDE.
### NEW
- مستكشف ملفات شجري مع عمليات: إنشاء/إعادة تسمية/حذف/تكرار.
- مخطّط المستند (Outline) وشريط تنقّل (Breadcrumb).
- بحث واستبدال على مستوى المشروع كامله.
- لوحة الأوامر، الفتح السريع، وبحث الرموز.
- تكامل Windows: نسخة واحدة، ربط الملفات، قائمة السياق، Jump List، وبروتوكول URI.
### IMPROVED
- تحكّم بإظهار/إخفاء اللوحات وأوضاع التركيز والقراءة.

- ملفات مفتوحة مؤخراً ومراقبة تغيّر الملفات خارجياً.
