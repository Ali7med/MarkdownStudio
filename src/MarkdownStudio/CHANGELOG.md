# Changelog

## [1.0.0] - 2026-07-18
### HIGHLIGHTS
- استوديو تحرير مرئي (WYSIWYG): حرّر مباشرة على المعاينة ويتولّد كود Markdown في الجهة الثانية.
### NEW
- شريط أدوات تحرير غني: عريض/مائل/يتوسّطه خط، عناوين، قوائم، اقتباس، كود، رابط، صورة، وحجم الخط.
- جداول في الوضع المرئي مع قائمة سياق لإضافة/حذف صف وعمود.
- مزامنة المؤشّر بين المعاينة ولوحة الكود.
- ميزة **ما الجديد** ونظام سجل التغييرات.
- دعم تعدّد اللغات مع الإنجليزية واتجاه نص تلقائي (RTL/LTR).
### IMPROVED
- نوافذ التنبيه تظهر متوسّطة على نافذة البرنامج مع تخطيط سليم للأزرار.
### FIXED
- إصلاح تداخل عنوان المستكشف مع أزراره.

## [0.9.0] - 2026-07-17
### HIGHLIGHTS
- مساحة عمل كاملة على مستوى المجلد بأسلوب IDE.
### NEW
- مستكشف ملفات شجري مع عمليات: إنشاء/إعادة تسمية/حذف/تكرار.
- مخطّط المستند (Outline) وشريط تنقّل (Breadcrumb).
- بحث واستبدال على مستوى المشروع كامله.
- لوحة الأوامر، الفتح السريع، وبحث الرموز.
- تكامل Windows: نسخة واحدة، ربط الملفات، قائمة السياق، Jump List، وبروتوكول URI.
### IMPROVED
- تحكّم بإظهار/إخفاء اللوحات وأوضاع التركيز والقراءة.

## [0.8.0] - 2026-07-17
### NEW
- تصدير إلى HTML وPDF وWord (DOCX) ونص عادي.
- شاشة ترحيب وحالة فارغة.
- تنبيه حفظ التغييرات عند الإغلاق.
- ملفات مفتوحة مؤخراً ومراقبة تغيّر الملفات خارجياً.
